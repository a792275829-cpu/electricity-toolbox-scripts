# 工具箱启动性能基准

测试环境：macOS，同一分支、同一 Python 虚拟环境，每种模式使用独立 Python 进程。

```text
首次测量：0.174603 秒（懒加载）/ 0.256319 秒（立即创建），缩短 31.9%
加入环境诊断后复测：0.194386 秒（懒加载）/ 0.289061 秒（立即创建），缩短 32.8%
```

命令：

```bash
python scripts/benchmark_toolbox_startup.py
python scripts/benchmark_toolbox_startup.py --eager
```

结果低于设计中的 40% 目标，但已经显著缩短启动时间。Tk 根窗口、工作台和任务中心本身构成固定成本；后续性能验收以不退化和保留实测记录为最低线。
