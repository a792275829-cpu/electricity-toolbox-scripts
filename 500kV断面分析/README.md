# 500kV断面分析

广东500kV节点电价清洗、断面识别与交互式拓扑网页生成模块。该项目已完整整理到电力工具箱仓库，日常使用优先从“电力工具箱 → 500kV断面分析”进入。

## 功能流程

1. 从内置拓扑的已确认连线端点生成500kV节点白名单。
2. 匹配数据源节点别名；同一拓扑节点存在多条记录时逐时取平均。
3. 检查相邻节点价格反向变化、两端价差严格超过阈值且连续至少4个时点的断面。
4. 清洗与断面明细在临时目录完成，最终只输出交互式HTML拓扑网页。
5. 网页颜色由累计触发时点和最大价差共同决定，越深表示异常越持久且价差越大。
6. 节点及断面电价曲线支持悬停或点按逐时查看时间和精确电价。

默认价差阈值为 `100元/MWh`，可在工具箱页面或命令行调整。

## 目录结构

```text
500kV断面分析/
├── run_500kv_analysis.py        # 可重复运行的批处理入口
├── 500kV节点拓扑.md             # 节点白名单与断面拓扑
├── 接线图.png                   # 节点定位底图
├── src/                         # 清洗、分析、网页生成源码
├── data/                        # 已确认节点坐标和定位候选数据
├── tools/                       # 节点位置校准工具
├── tests/                       # 原项目完整单元测试
├── docs/                        # 设计与历史实施文档
└── local_data/                  # 原始数据及历史结果，仅保存在本机
```

`local_data/` 已加入仓库忽略规则，避免真实业务数据和生成报表进入版本控制。

## 工具箱运行

从仓库根目录双击 `电力工具箱.command`，选择侧栏的“500kV断面分析”：

1. 添加一份或多份节点电价 `.xlsx`。
2. 确认内置拓扑和输出目录。
3. 设置价差阈值，点击“开始分析”。
4. 完成后可直接打开结果目录或最近生成的网页。

也可以双击本目录的 `500kV 断面分析.command` 打开兼容的独立窗口。

## 命令行运行

在仓库根目录执行：

```bash
.venv/bin/python "500kV断面分析/run_500kv_analysis.py" \
  --output-root ~/Downloads/500kV断面分析 \
  --threshold 100 \
  /path/to/日前节点电价查询.xlsx
```

可以一次传入多个Excel；每份输入最终只保留一个分析网页。

## 输出

每份输入数据会在输出根目录生成一个同名子目录：

- `原数据文件名_500kV分析.html`

清洗Excel和断面明细Excel只作为临时计算文件，生成网页后自动删除；原始工作簿不会被覆盖。

## 节点位置校准

双击 `校准500kV节点位置.command` 可打开本地校准页面。页面读取：

- `接线图.png`
- `data/500kv_node_position_candidates.json`
- `data/500kv_node_positions.json`

如需重新从OCR记录生成候选位置，将数据放到 `local_data/ocr-records.json`，或通过环境变量 `SPOTMAP_OCR_DATA` 指定，并确保当前环境已安装可选依赖 `opencv-python`。
